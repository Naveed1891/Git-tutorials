Here are some commonly used basic Git commands in Git Bash:

git init: Initialize a new Git repository.
git add <filename>: Add a specific file to the staging area.
git add .: Add all changes in the current directory to the staging area.
git commit -m "<commit message>": Commit the changes in the staging area with a brief message describing the changes.
git status: Check the status of your repository, including any changes made since the last commit.
git log: Display the commit history for the current branch.
git branch: Display a list of all branches in the repository.
git branch <branch-name>: Create a new branch with the specified name.
git checkout <branch-name>: Switch to the specified branch.
git merge <branch-name>: Merge the specified branch into the current branch.
git push -u origin master: Push the changes in the current branch to the specified remote branch.
git pull origin master: Pull the latest changes from the specified remote branch and merge them into the current branch.
